
# Question 159




    Create sample programs to study the values of these expressions:

\beginitemize
\item `sizeof(void)`
\item  `sizeof(0)`
\item `sizeof('x')`
\item `sizeof("hello")`
\enditemize 


# Answer



```c
sizeof(void)    = 1
sizeof(0)       = 4
sizeof('x')     = 4
sizeof("hello") = 6
```
